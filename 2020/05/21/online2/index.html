<<NTCIR-15 QA Lab-PoliInfo-2>>

フェイクニュースなどの社会問題において政治家の発言の信憑性を判断するためには、発言者の政治的スタンス、文脈を考慮した要約、根拠となる一次情報の特定といった課題を解決する必要があります。
NTCIR-15 QA Lab-PoliInfo2 ( ./
)では、自然言語処理のアプローチにより、これらの政治課題を解決するために、3つのサブタスクを設計しました。

1. Stance Classification :  ./stance-classification-task/
2. Dialog Summarization : ./dialog-summarization-task/
3. Entity Linking : ./entity-linking-task/
